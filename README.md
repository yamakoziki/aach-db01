# 登山データ抽出ツール

## 使用方法

### 基本的な使用方法
```bash
python mountain_extractor.py data.csv "田中太郎"
```

### CSVファイルに出力
```bash
python mountain_extractor.py data.csv "田中太郎" -o 田中太郎_登山記録.csv
```

## CSVファイル形式

入力CSVファイルには以下のカラムが必要です：
- 年度
- 暦年  
- 開始日
- 終了日
- 行動
- 停滞
- メンバー
- 山域
- シーズン
- ルート・特記事項
- 出典
- リンク
- 種別

## 出力形式

抽出されるデータには以下のカラムが含まれます：
- 年度、暦年、開始日、終了日、行動、停滞、メンバー、山域、シーズン、ルート・特記事項、出典
